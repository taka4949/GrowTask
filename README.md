GrowTask (育成型タスク管理アプリ)

アプリ概要
「タスク管理」に「育成ゲーム」の要素を組み合わせたAndroidアプリです。
タスクを完了すると経験値が得られ、木が成長していくことで、ユーザーの「継続」を楽しくサポートします。

開発の背景・目的
自身の習慣化を助けるツールとして開発しました。
ただのToDoリストではモチベーションが続かない課題を解決するため、「成長の可視化」をテーマにしています。

技術スタック
言語: Kotlin (1.9.22)
データベース: Realm Database (MongoDB Realm Kotlin SDK)
非同期処理: Kotlin Coroutines
アーキテクチャ: MVVM (学習中)
ツール: Android Studio, Git/GitHub

現在の開発状況と今後の展望
【開発ステータス：プロトタイプ】
現在は基本機能（タスク追加・完了・レベルアップ演出・日付管理）の実装が完了した段階です。

【今後のロードマップ】
実務レベルの技術力を身につけるため、以下の機能を毎日少しずつ実装・改良しています。

1.  バックエンド連携 TypeScriptを用いたAPIサーバーの構築（学習中）
2.  UI/UXの改善 アニメーション追加や直感的な操作性の向上
3.  ゲーミフィケーション強化 成長段階の追加、ポイント交換機能など


本プロジェクトは、プログラミング学習開始から約4ヶ月で作成した初めてのAndroidアプリです。
至らない点も多々あるかと思いますが、「動くものを作り、使いながら改善する」プロセスを大切にしています。
2026年4月のリリースを目指しています。
